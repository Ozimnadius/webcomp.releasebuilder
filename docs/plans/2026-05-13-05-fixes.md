## Задача 1: Admin-only access control

Контроллер сейчас доступен любому авторизованному пользователю. Нужно добавить проверку `IsAdmin()` через отдельный Bitrix ActionFilter, чтобы не дублировать логику в каждом action-методе.

- Изменить: `lib/controller/releasebuilder.php`

- [ ] **Шаг 1.1: Создать класс AdminAccess**

Создать файл `lib/ActionFilter/AdminAccess.php`:

```php
<?php

namespace Webcomp\ReleaseBuilder\ActionFilter;

use Bitrix\Main\Engine\ActionFilter\Base;
use Bitrix\Main\Error;
use Bitrix\Main\Event;
use Bitrix\Main\EventResult;

/**
 * Запрещает доступ к action-методам контроллера для неадминистраторов.
 *
 * Использует глобальный объект $USER (CUser), стандартный для Bitrix.
 * Возвращает HTTP 403 и добавляет ошибку в ответ контроллера.
 */
class AdminAccess extends Base
{
    public function onBeforeAction(Event $event)
    {
        global $USER;

        if (!$USER instanceof \CUser || !$USER->IsAdmin()) {
            $this->addError(new Error('Доступ запрещён: требуются права администратора', 403));
            return new EventResult(EventResult::ERROR, null, null, $this);
        }

        return null;
    }
}
```

```php
/**
 * Генерирует install/version.php внутри staging-директории версии.
 *
 * Файл содержит массив $arModuleVersion с VERSION и VERSION_DATE
 * в формате, ожидаемом установщиком модулей Bitrix.
 *
 * @param string $module  Идентификатор модуля (например, 'webcomp.market').

}
```

```php
     * что исключает потерю данных при сбое между двумя операциями.
     *
     * @param string   $module       Идентификатор модуля.
     * @param string[] $patterns     Паттерны исключений. Ключи массива сбрасываются.
     * @param string   $templateName Название шаблона (пустая строка — очистить).
     * @throws \RuntimeException Если config.json содержит невалидный JSON или файл не удалось записать.
     */
    public function saveConfig(string $module, array $patterns, string $templateName): void
    {
        $data          = $this->readAllData();
        $data[$module] = [
            'patterns'     => array_values($patterns),
            'templateName' => $templateName,
        ];

        $encoded = json_encode($data, JSON_PRETTY_PRINT | JSON_UNESCAPED_UNICODE | JSON_THROW_ON_ERROR);
        $this->filesystem->putContents($this->configPath, $encoded);
    }
```
